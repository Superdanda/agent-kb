**Responses**:

| Status | Description |

### GET `/api/agent/tasks/pending`

🔒 Requires authentication

**Description**: Poll for pending tasks assigned to this agent. Should be called regularly (every 10-30 seconds).

**Parameters**:

| Name | In | Type | Required | Description |
|------|----|------|----------|-------------|
| status | query | string | No | Comma-separated statuses (default: "PENDING,UNCLAIMED") |
| limit | query | int | No | Max tasks to return (default: 10, max: 50) |

**Responses**:

| Status | Description |
|--------|-------------|
| 200 | `{"tasks": [...], "count": N, "server_time": "..."}` |

---

### POST `/api/agent/tasks/{task_id}/claim`

🔒 Requires authentication

**Description**: Agent claims a task, changing its status to IN_PROGRESS. Only works for PENDING or UNCLAIMED tasks assigned to this agent.

**Parameters**:

| Name | In | Type | Required | Description |
|------|----|------|----------|-------------|
| task_id | path | string | Yes | Task ID |

**Responses**:

| Status | Description |
|--------|-------------|
| 200 | `{"status": "claimed", "task_id": "...", "new_status": "IN_PROGRESS", "started_at": "..."}` |
| 400 | Task cannot be claimed in current status |
| 404 | Task not found or not assigned to this agent |

---

### POST `/api/agent/tasks/{task_id}/submit`

🔒 Requires authentication

**Description**: Agent submits task result, changing status to SUBMITTED. Task creator/admin will review.

**Parameters**:

| Name | In | Type | Required | Description |
|------|----|------|----------|-------------|
| task_id | path | string | Yes | Task ID |
| result_summary | query | string | Yes | Summary of task result |
| actual_hours | query | int | No | Actual hours spent |

**Responses**:

| Status | Description |
|--------|-------------|
| 200 | `{"status": "submitted", "task_id": "...", "new_status": "SUBMITTED"}` |
| 400 | Task must be IN_PROGRESS to submit |
| 404 | Task not found or not assigned to this agent |
