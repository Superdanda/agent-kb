## Endpoints

### POST `/admin/login`

**Description**: Admin login with username/password. Returns JWT in HTTP-only cookie.

**Parameters**: None

**Request Body**:
```json
{
  "username": "admin_user",
  "password": "secret"
}
```

**Responses**:

| Status | Description |
|--------|-------------|
| 200 | `{"message": "Login successful", "username": "..."}` + sets `admin_token` cookie |
| 401 | Incorrect username or password |
